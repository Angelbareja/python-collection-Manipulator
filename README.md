# 🎓 Student Data Organizer

A simple Python-based console application to manage student records efficiently. This project demonstrates the use of Python collections (lists), CRUD operations, and menu-driven programming.

## 📌 Project Overview

The **Student Data Organizer** helps users store and manage student information such as:

- Student ID
- Name
- Age
- Grade
- Date of Birth
- Subjects

It provides an interactive command-line interface for performing student data management tasks.

## ⚙ How It Works

### 1. Add Student

Users can enter:

* Student ID
* Name
* Age
* Grade
* Date of Birth
* Subjects

### 2. Display Students

Shows all saved student records.

### 3. Update Student

Search student by ID and update details.

### 4. Delete Student

Remove a student record by ID.

### 5. Display Subjects

Displays all subjects from stored records.

### 6. Exit
